# Affine Photogrammetry Transformation (MATLAB)

This repository provides a clean MATLAB implementation of 2D affine
transformation for photogrammetric and geospatial coordinate mapping.

## Mathematical Model
X = A x + t

Affine parameters are estimated using least squares adjustment.

## Features
- 2D affine transformation
- Least squares estimation
- RMSE evaluation
- Ready-to-use demo script

## Folder Structure
- src/    : core affine functions
- demo/   : example script
- data/   : sample input data

## How to Run
1. Open MATLAB
2. Navigate to demo folder
3. Run `demo_affine.m`
